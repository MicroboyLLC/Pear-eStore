Pear eStore Client Installation Instructions

To install and run the Pear eStore client, please follow these steps:

  1. Install Python 3.10.5.
  2. Download the PearStore.zip file here.
  3. Extract the contents of the PearStore.zip file to a location of your choice.
  4. Open a Terminal or Command Prompt window and navigate to the extracted directory.
  5. Run the following command to install the necessary libraries:
    6. pip install -r requirements.txt
  6. Open the PearStore.cpython-310.pyc file with the Python Launcher by either double-clicking it or right-clicking it and selecting "Open with" and then "Python Launcher".
  7. On startup, the client will ask you to enter your Solana wallet address. Enter your Solana wallet address and press Enter.
  8. You can now use the client to shop for digital goods.
  9. To change your Solana wallet address, open the client and type "settings". Then, type "solana address" and enter your new Solana wallet address. Finally, type "exit settings" to save the changes and return to the main command prompt.

Enjoy shopping at Pear eStore!
